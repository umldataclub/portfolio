# Blockchain Analysis

> An official codebase hasn't been created yet for this project, but this page outlines what this project could look like

# Vision
Building foundational pipelines for exploratory data analysis (EDA) of blockchain data and prediction market dynamics.

### Data Collection
- **Polymarket Trades**: Automated fetching of trade data.
- **Funding Information**: Tracing the chain of funders for specific traders.
- **Analytics**: Correlating funding patterns with market outcomes.

### Technologies
- APIs for data retrieval.
- Exploratory Data Analysis (EDA) studies or tools.
- Storage solutions for large-scale datasets.

## Roadmap
- [] Initial API integration
- [] Basic trade decoding
- [ ] Advanced visualization of funding networks
- [ ] Anomaly detection in market behavior
